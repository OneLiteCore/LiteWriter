Markdown एक सरल और हल्का लेखन प्रारूप है। यह आपको साधारण टेक्स्ट का उपयोग करके शीर्षक, जोर, सूचियाँ और चित्र जैसे कई सामान्य rich-text रूपों को व्यक्त करने देता है, बिना भारी दस्तावेज़ संपादकों या Word जैसी rich-text दस्तावेज़ फ़ॉर्मैटों पर निर्भर हुए। यह पृष्ठ स्वयं भी Markdown में लिखा गया है।

# Markdown सिंटैक्स कैसे काम करता है

Markdown को रेंडर होने से पहले भी आसानी से पढ़ा जा सकता है। नीचे कुछ सामान्य सिंटैक्स उदाहरण दिए गए हैं:

- `# Heading` एक शीर्षक बनाता है। छोटे स्तर के शीर्षकों के लिए और अधिक `#` चिह्नों का उपयोग किया जा सकता है।
- `---` एक क्षैतिज विभाजक रेखा बनाता है।
- `**bold**` पाठ को **बोल्ड** बनाता है।
- `*italic*` पाठ को *इटैलिक* बनाता है।
- `![Name](Image URL link)` चित्र डालने का सिंटैक्स है।
- `- item` एक सूची आइटम बनाता है।
- `~~text~~` स्ट्राइकथ्रू जोड़ता है, जैसे ~~पुराना पाठ~~।

क्योंकि इसका सिंटैक्स बहुत छोटा और सीधा है, Markdown लेखकों को फ़ॉर्मैटिंग टूल्स के बजाय सामग्री पर ध्यान केंद्रित करने में मदद करता है।

# टेक्स्ट फ़ाइलें क्यों

साधारण टेक्स्ट, टेक्स्ट को संग्रहीत करने का सबसे सरल फ़ॉर्मैट है, और यही कारण है कि इसका इतना व्यापक उपयोग होता है। लगभग सभी लेखन सॉफ़्टवेयर, क्लाउड नोट टूल, ऑफिस सूट और यहाँ तक कि कमांड-लाइन टूल भी टेक्स्ट फ़ाइलों को पढ़, लिख और उपयोग कर सकते हैं।

दशकों पहले DOS सिस्टम पर लिखी गई टेक्स्ट फ़ाइलें आज भी Android पर सामान्य रूप से पढ़ी और संपादित की जा सकती हैं। इसी तरह, आज Android पर लिखी गई टेक्स्ट फ़ाइलें भी बहुत संभव है कि आने वाले दशकों में भविष्य के कंप्यूटरों द्वारा पढ़ी और लिखी जा सकें।

यह ऐप आपके लेखन को `.txt` टेक्स्ट फ़ाइलों के रूप में सहेजता है, जिससे आपका डेटा अधिक टिकाऊ और आसानी से स्थानांतरित करने योग्य बनता है। यदि आप कभी अपनी सभी नोट्स को किसी अन्य क्लाउड नोटबुक, नॉलेज बेस या लेखन टूल में ले जाना चाहें, तो माइग्रेशन बहुत आसान हो जाएगा और अक्सर एक ही चरण में किया जा सकेगा।
