Markdown یک قالب نوشتاری ساده و سبک است. این قالب به شما اجازه می‌دهد با استفاده از متن ساده، بسیاری از شکل‌های رایج متن غنی مانند عنوان‌ها، تاکید، فهرست‌ها و تصویرها را بیان کنید، بدون آن‌که به ویرایشگرهای سنگین اسناد یا قالب‌های متن غنی شبیه Word وابسته باشید. خود این صفحه نیز با Markdown نوشته شده است.

# نحو Markdown چگونه کار می‌کند

Markdown حتی پیش از رندر شدن هم خوانا است. در ادامه چند نمونه رایج از نحو آن آمده است:

- `# Heading` یک عنوان ایجاد می‌کند. برای سطوح پایین‌تر عنوان می‌توان از تعداد بیشتری `#` استفاده کرد.
- `---` یک خط جداکننده افقی ایجاد می‌کند.
- `**bold**` متن را **پررنگ** می‌کند.
- `*italic*` متن را *مورب* می‌کند.
- `![Name](Image URL link)` نحوِ وارد کردن تصویر است.
- `- item` یک مورد فهرست ایجاد می‌کند.
- `~~text~~` خط خوردگی اضافه می‌کند، مانند ~~متن قدیمی~~.

چون این نحو بسیار کوچک و مستقیم است، Markdown به نویسندگان کمک می‌کند به‌جای ابزارهای قالب‌بندی، روی محتوا تمرکز کنند.

# چرا فایل‌های متنی

متن ساده ساده‌ترین قالب برای ذخیره‌سازی متن است و دقیقاً به همین دلیل تا این حد فراگیر شده است. تقریباً همه نرم‌افزارهای نوشتن، ابزارهای یادداشت ابری، مجموعه‌های اداری و حتی ابزارهای خط فرمان می‌توانند فایل‌های متنی را بخوانند، بنویسند و استفاده کنند.

فایل‌های متنی‌ای که دهه‌ها پیش در سیستم‌های DOS نوشته شده‌اند، امروز هم در Android به‌طور عادی خوانده و ویرایش می‌شوند. به همین شکل، بسیار محتمل است که فایل‌های متنی‌ای که امروز در Android نوشته می‌شوند، دهه‌ها بعد نیز در رایانه‌های آینده همچنان قابل خواندن و نوشتن باشند.

این برنامه نوشته‌های شما را به‌صورت فایل‌های متنی `.txt` ذخیره می‌کند و همین موضوع باعث می‌شود داده‌های شما ماندگارتر و قابل‌انتقال‌تر باشند. اگر روزی بخواهید همه یادداشت‌های خود را به یک دفترچه ابری دیگر، پایگاه دانش یا ابزار نوشتن دیگری منتقل کنید، این مهاجرت بسیار آسان‌تر خواهد بود و اغلب در یک مرحله انجام می‌شود.
