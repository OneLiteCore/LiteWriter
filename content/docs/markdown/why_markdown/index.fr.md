Markdown est un format d’écriture simple et léger. Il vous permet d’utiliser du texte brut pour exprimer de nombreux styles de mise en forme courants, comme les titres, l’emphase, les listes et les images, sans dépendre d’éditeurs de documents lourds ou de formats de documents enrichis de type Word. Cette page elle-même est également écrite en Markdown.

# Comment fonctionne la syntaxe Markdown

Markdown est facile à lire même avant d’être rendu. Voici quelques exemples de syntaxe courants :

- `# Heading` crée un titre. Davantage de symboles `#` peuvent être utilisés pour des niveaux de titre inférieurs.
- `---` crée une ligne de séparation horizontale.
- `**bold**` met le texte en **gras**.
- `*italic*` met le texte en *italique*.
- `![Name](Image URL link)` est la syntaxe pour insérer une image.
- `- item` crée un élément de liste.
- `~~text~~` ajoute un texte barré, comme ~~ancien texte~~.

Comme la syntaxe est très petite et directe, Markdown aide les auteurs à se concentrer sur le contenu plutôt que sur les outils de mise en forme.

# Pourquoi les fichiers texte

Le texte brut est le format le plus simple pour stocker du texte, et c’est précisément pour cela qu’il est si largement utilisé. Presque tous les logiciels d’écriture, outils de notes dans le cloud, suites bureautiques et même outils en ligne de commande peuvent lire, écrire et utiliser des fichiers texte.

Les fichiers texte écrits il y a plusieurs décennies sur des systèmes DOS peuvent encore être lus et modifiés normalement sur Android aujourd’hui. De la même manière, il est très probable que les fichiers texte écrits aujourd’hui sur Android puissent encore être lus et écrits sur les ordinateurs du futur dans plusieurs décennies.

Cette application enregistre vos écrits sous forme de fichiers texte `.txt`, ce qui rend vos données plus durables et plus portables. Si vous souhaitez un jour déplacer toutes vos notes vers un autre carnet cloud, une base de connaissances ou un autre outil d’écriture, la migration sera bien plus simple et pourra souvent se faire en une seule étape.
