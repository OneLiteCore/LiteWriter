Markdown é um formato de escrita simples e leve. Ele permite usar texto simples para expressar muitos estilos comuns de rich text, como títulos, ênfase, listas e imagens, sem depender de editores de documentos pesados ou de formatos de documento rich text no estilo do Word. Esta própria página também foi escrita em Markdown.

# Como a sintaxe Markdown funciona

Markdown é fácil de ler mesmo antes de ser renderizado. Abaixo estão alguns exemplos comuns de sintaxe:

- `# Heading` cria um título. Mais símbolos `#` podem ser usados para níveis de título inferiores.
- `---` cria uma linha divisória horizontal.
- `**bold**` deixa o texto em **negrito**.
- `*italic*` deixa o texto em *itálico*.
- `![Name](Image URL link)` é a sintaxe para inserir uma imagem.
- `- item` cria um item de lista.
- `~~text~~` adiciona tachado, como em ~~texto antigo~~.

Como a sintaxe é tão pequena e direta, Markdown ajuda quem escreve a se concentrar no conteúdo em vez de nas ferramentas de formatação.

# Por que arquivos de texto

Texto simples é o formato mais simples para armazenar texto, e é exatamente por isso que ele é tão amplamente usado. Quase todos os aplicativos de escrita, ferramentas de notas em nuvem, suítes de escritório e até ferramentas de linha de comando conseguem ler, escrever e trabalhar com arquivos de texto.

Arquivos de texto escritos décadas atrás em sistemas DOS ainda podem ser lidos e editados normalmente no Android hoje. Da mesma forma, é muito provável que arquivos de texto escritos hoje no Android continuem podendo ser lidos e gravados em computadores do futuro daqui a muitas décadas.

Este aplicativo salva sua escrita como arquivos de texto `.txt`, o que torna seus dados mais duráveis e portáteis. Se algum dia você quiser mover todas as suas notas para outro caderno em nuvem, base de conhecimento ou outra ferramenta de escrita, a migração será muito mais fácil e muitas vezes poderá ser feita em uma única etapa.
