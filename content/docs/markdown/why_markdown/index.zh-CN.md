Markdown 是一种简约、轻量的写作格式。它允许你只用纯文本来表达许多常见的富文本样式，例如标题、强调、列表和图片，而不必依赖沉重的文档编辑器或类似 Word 的富文本文件格式。本文本身也是使用 Markdown 编写的。

# Markdown 语法如何工作

Markdown 即使在渲染之前也很容易阅读。下面是一些常见语法示例：

- `# Heading` 用于创建标题。使用更多的 `#` 符号可以表示更低级别的标题。
- `---` 用于创建分割线。
- `**bold**` 会让文本显示为 **粗体**。
- `*italic*` 会让文本显示为 *斜体*。
- `![Name](Image URL link)` 是插入图片的语法。
- `- item` 用于创建列表项。
- `~~text~~` 会添加删除线，例如 ~~旧文本~~。

正因为它的语法非常简洁直接，Markdown 能帮助写作者把注意力放在内容本身，而不是排版工具上。

# 为什么使用文本文件

纯文本是用于存储文字内容最简单的格式，也正因如此，它才会被如此广泛地使用。几乎所有写作软件、云笔记工具、办公套件，甚至命令行工具，都可以读写和使用文本文件。

数十年前在 DOS 系统中写下的文本文件，今天依然可以在 Android 上被正常读取和编辑。同样地，今天在 Android 上写下的文本文件，在数十年后的未来计算机上，也极有可能依然可以被读取和写入。

本应用将你的写作内容保存为 `.txt` 文本文件，这让你的数据更耐久，也更容易迁移。如果未来某一天你想把所有笔记迁移到其他云笔记、知识库或写作工具中，整个迁移过程都会简单得多，而且通常可以一步完成。
