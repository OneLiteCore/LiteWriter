Markdown ist ein einfaches und leichtgewichtiges Schreibformat. Es ermöglicht dir, mit Klartext viele gängige Rich-Text-Formatierungen wie Überschriften, Hervorhebungen, Listen und Bilder auszudrücken, ohne auf schwere Dokumenteditoren oder Word-ähnliche Rich-Text-Dokumentformate angewiesen zu sein. Auch diese Seite selbst ist in Markdown geschrieben.

# Wie Markdown-Syntax funktioniert

Markdown ist schon vor dem Rendern leicht zu lesen. Hier sind einige häufige Syntaxbeispiele:

- `# Heading` erstellt eine Überschrift. Mit mehreren `#`-Zeichen lassen sich niedrigere Überschriftsebenen erzeugen.
- `---` erstellt eine horizontale Trennlinie.
- `**bold**` macht Text **fett**.
- `*italic*` macht Text *kursiv*.
- `![Name](Image URL link)` ist die Syntax zum Einfügen eines Bildes.
- `- item` erstellt einen Listeneintrag.
- `~~text~~` fügt eine Durchstreichung hinzu, zum Beispiel ~~alter Text~~.

Weil die Syntax so klein und direkt ist, hilft Markdown Schreibenden, sich auf den Inhalt statt auf Formatierungswerkzeuge zu konzentrieren.

# Warum Textdateien

Klartext ist das einfachste Format zum Speichern von Text, und genau deshalb ist es so weit verbreitet. Fast alle Schreibprogramme, Cloud-Notiz-Tools, Office-Suiten und sogar Kommandozeilenwerkzeuge können Textdateien lesen, schreiben und verwenden.

Textdateien, die vor Jahrzehnten auf DOS-Systemen geschrieben wurden, lassen sich heute auf Android immer noch ganz normal lesen und bearbeiten. Genauso ist es sehr wahrscheinlich, dass Textdateien, die heute auf Android geschrieben werden, auch in Jahrzehnten noch von zukünftigen Computern gelesen und geschrieben werden können.

Diese App speichert deine Texte als `.txt`-Dateien, was deine Daten langlebiger und portabler macht. Wenn du deine Notizen später einmal in ein anderes Cloud-Notizbuch, eine Wissensdatenbank oder ein anderes Schreibwerkzeug übertragen möchtest, ist die Migration viel einfacher und oft in einem Schritt möglich.
