Markdown es un formato de escritura simple y ligero. Te permite usar texto plano para expresar muchos estilos comunes de texto enriquecido, como encabezados, énfasis, listas e imágenes, sin depender de editores de documentos pesados ni de formatos de texto enriquecido al estilo de Word. Esta misma página también está escrita en Markdown.

# Cómo funciona la sintaxis de Markdown

Markdown es fácil de leer incluso antes de renderizarse. A continuación se muestran algunos ejemplos comunes de sintaxis:

- `# Heading` crea un encabezado. Se pueden usar más símbolos `#` para niveles de encabezado inferiores.
- `---` crea una línea divisoria horizontal.
- `**bold**` pone el texto en **negrita**.
- `*italic*` pone el texto en *cursiva*.
- `![Name](Image URL link)` es la sintaxis para insertar una imagen.
- `- item` crea un elemento de lista.
- `~~text~~` añade texto tachado, como ~~texto antiguo~~.

Como la sintaxis es tan pequeña y directa, Markdown ayuda a los escritores a centrarse en el contenido en lugar de en las herramientas de formato.

# Por qué los archivos de texto

El texto plano es el formato más simple para almacenar texto, y precisamente por eso se usa tanto. Casi todas las aplicaciones de escritura, herramientas de notas en la nube, suites ofimáticas e incluso herramientas de línea de comandos pueden leer, escribir y trabajar con archivos de texto.

Los archivos de texto escritos hace décadas en sistemas DOS todavía pueden leerse y editarse con normalidad en Android hoy en día. Del mismo modo, es muy probable que los archivos de texto escritos hoy en Android sigan pudiendo leerse y escribirse en las computadoras del futuro dentro de varias décadas.

Esta aplicación guarda tu escritura como archivos de texto `.txt`, lo que hace que tus datos sean más duraderos y portables. Si alguna vez quieres mover todas tus notas a otro cuaderno en la nube, base de conocimiento u otra herramienta de escritura, la migración será mucho más fácil y a menudo podrá hacerse en un solo paso.
