Markdown è un formato di scrittura semplice e leggero. Ti permette di usare testo semplice per esprimere molti stili comuni di testo formattato, come titoli, enfasi, elenchi e immagini, senza dipendere da editor di documenti pesanti o da formati di documenti rich text in stile Word. Anche questa pagina è scritta in Markdown.

# Come funziona la sintassi Markdown

Markdown è facile da leggere anche prima del rendering. Di seguito trovi alcuni esempi comuni di sintassi:

- `# Heading` crea un titolo. È possibile usare più simboli `#` per livelli di titolo inferiori.
- `---` crea una linea divisoria orizzontale.
- `**bold**` rende il testo **in grassetto**.
- `*italic*` rende il testo *in corsivo*.
- `![Name](Image URL link)` è la sintassi per inserire un'immagine.
- `- item` crea una voce di elenco.
- `~~text~~` aggiunge una barratura, come ~~testo vecchio~~.

Poiché la sintassi è così piccola e diretta, Markdown aiuta chi scrive a concentrarsi sul contenuto invece che sugli strumenti di formattazione.

# Perché i file di testo

Il testo semplice è il formato più semplice per memorizzare testo, ed è proprio per questo che è così ampiamente usato. Quasi tutti i software di scrittura, gli strumenti di note cloud, le suite per ufficio e persino gli strumenti da riga di comando possono leggere, scrivere e usare file di testo.

I file di testo scritti decenni fa sui sistemi DOS possono ancora essere letti e modificati normalmente su Android oggi. Allo stesso modo, è molto probabile che i file di testo scritti oggi su Android rimangano leggibili e scrivibili anche sui computer del futuro tra molti decenni.

Questa app salva i tuoi scritti come file di testo `.txt`, rendendo i tuoi dati più durevoli e portabili. Se un giorno vorrai spostare tutte le tue note in un altro taccuino cloud, una base di conoscenza o un altro strumento di scrittura, la migrazione sarà molto più semplice e spesso potrà essere eseguita in un solo passaggio.
