Ang Markdown ay isang simple at magaang na format sa pagsusulat. Hinahayaan ka nitong gumamit ng plain text upang maipahayag ang maraming karaniwang rich-text style gaya ng mga heading, emphasis, listahan, at mga larawan, nang hindi umaasa sa mabibigat na document editor o Word-style rich-text document formats. Ang pahinang ito mismo ay nakasulat din sa Markdown.

# Paano gumagana ang Markdown syntax

Madaling basahin ang Markdown kahit bago pa ito i-render. Narito ang ilang karaniwang halimbawa ng syntax:

- Ang `# Heading` ay lumilikha ng heading. Mas maraming `#` ang maaaring gamitin para sa mas mababang antas ng heading.
- Ang `---` ay lumilikha ng pahalang na guhit na naghihiwalay.
- Ang `**bold**` ay ginagawang **bold** ang teksto.
- Ang `*italic*` ay ginagawang *italic* ang teksto.
- Ang `![Name](Image URL link)` ang syntax para magpasok ng larawan.
- Ang `- item` ay lumilikha ng item sa listahan.
- Ang `~~text~~` ay naglalagay ng strikethrough, gaya ng ~~lumang teksto~~.

Dahil napakaliit at direkta ng syntax, tinutulungan ng Markdown ang mga manunulat na tumuon sa nilalaman sa halip na sa mga formatting tool.

# Bakit mga text file

Ang plain text ang pinakasimpleng format para sa pag-iimbak ng teksto, at iyan mismo ang dahilan kung bakit ito malawakang ginagamit. Halos lahat ng writing software, cloud note tools, office suite, at maging command-line tools ay kayang magbasa, magsulat, at gumamit ng mga text file.

Ang mga text file na isinulat ilang dekada na ang nakalipas sa mga DOS system ay maaari pa ring basahin at i-edit nang normal sa Android ngayon. Sa parehong paraan, malaki ang posibilidad na ang mga text file na isinusulat sa Android ngayon ay mababasa at maisusulat pa rin ng mga computer sa hinaharap pagkalipas ng maraming dekada.

Iniimbak ng app na ito ang iyong mga sinusulat bilang mga `.txt` text file, kaya mas matibay at mas madaling ilipat ang iyong data. Kung gusto mong ilipat balang araw ang lahat ng iyong tala sa ibang cloud notebook, knowledge base, o writing tool, mas magiging madali ang migration at madalas ay magagawa ito sa isang hakbang.
