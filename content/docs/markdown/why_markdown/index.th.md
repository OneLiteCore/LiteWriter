Markdown คือรูปแบบการเขียนที่เรียบง่ายและมีน้ำหนักเบา ช่วยให้คุณใช้ข้อความล้วนเพื่อแสดงรูปแบบ rich text ที่ใช้กันทั่วไปได้มากมาย เช่น หัวข้อ การเน้น รายการ และรูปภาพ โดยไม่ต้องพึ่งโปรแกรมแก้ไขเอกสารขนาดใหญ่หรือรูปแบบเอกสาร rich text แบบเดียวกับ Word หน้านี้เองก็เขียนด้วย Markdown เช่นกัน

# ไวยากรณ์ Markdown ทำงานอย่างไร

Markdown อ่านได้ง่ายแม้ก่อนจะถูกเรนเดอร์ ด้านล่างคือตัวอย่างไวยากรณ์ที่ใช้บ่อย:

- `# Heading` ใช้สร้างหัวข้อ และสามารถใช้สัญลักษณ์ `#` เพิ่มขึ้นเพื่อสร้างหัวข้อในระดับที่ย่อยลงไปได้
- `---` ใช้สร้างเส้นแบ่งแนวนอน
- `**bold**` ทำให้ข้อความเป็น **ตัวหนา**
- `*italic*` ทำให้ข้อความเป็น *ตัวเอียง*
- `![Name](Image URL link)` คือไวยากรณ์สำหรับแทรกรูปภาพ
- `- item` ใช้สร้างรายการ
- `~~text~~` ใช้เพิ่มข้อความขีดทับ เช่น ~~ข้อความเก่า~~

เพราะไวยากรณ์มีขนาดเล็กและตรงไปตรงมา Markdown จึงช่วยให้นักเขียนโฟกัสกับเนื้อหาแทนที่จะต้องเสียสมาธิกับเครื่องมือจัดรูปแบบ

# ทำไมต้องใช้ไฟล์ข้อความ

ข้อความล้วนคือรูปแบบที่ง่ายที่สุดสำหรับการเก็บข้อความ และนั่นเองคือเหตุผลที่มันถูกใช้อย่างแพร่หลาย ซอฟต์แวร์เขียนข้อความ เครื่องมือจดบันทึกบนคลาวด์ ชุดโปรแกรมสำนักงาน และแม้แต่เครื่องมือบรรทัดคำสั่ง เกือบทั้งหมดสามารถอ่าน เขียน และใช้งานไฟล์ข้อความได้

ไฟล์ข้อความที่ถูกเขียนขึ้นเมื่อหลายสิบปีก่อนบนระบบ DOS ยังคงสามารถเปิดอ่านและแก้ไขได้ตามปกติบน Android ในปัจจุบัน เช่นเดียวกัน ไฟล์ข้อความที่เขียนบน Android วันนี้ก็มีแนวโน้มสูงมากว่าจะยังคงถูกอ่านและเขียนได้บนคอมพิวเตอร์ในอนาคตอีกหลายสิบปีข้างหน้า

แอปนี้บันทึกงานเขียนของคุณเป็นไฟล์ข้อความ `.txt` ทำให้ข้อมูลของคุณคงทนและย้ายต่อได้ง่ายขึ้น หากวันหนึ่งคุณต้องการย้ายบันทึกทั้งหมดไปยังสมุดบันทึกบนคลาวด์ ฐานความรู้ หรือเครื่องมือเขียนอื่น การย้ายข้อมูลจะง่ายกว่ามาก และมักทำได้ในขั้นตอนเดียว
