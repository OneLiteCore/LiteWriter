Markdown は、シンプルで軽量な文章作成フォーマットです。見出し、強調、リスト、画像など、よく使われる多くのリッチテキスト表現を、重い文書エディタや Word のようなリッチテキスト文書形式に頼らず、プレーンテキストだけで表現できます。このページ自体も Markdown で書かれています。

# Markdown 構文のしくみ

Markdown は、レンダリングされる前でも読みやすいのが特長です。以下はよく使われる構文の例です。

- `# Heading` は見出しを作成します。`#` を増やすことで、より下位の見出しレベルを表せます。
- `---` は区切り線を作成します。
- `**bold**` は文字を **太字** にします。
- `*italic*` は文字を *斜体* にします。
- `![Name](Image URL link)` は画像を挿入するための構文です。
- `- item` はリスト項目を作成します。
- `~~text~~` は ~~古いテキスト~~ のように取り消し線を追加します。

構文がとても小さく直接的なので、Markdown は書き手が書式設定ツールではなく内容そのものに集中するのを助けます。

# テキストファイルを使う理由

プレーンテキストは、文字を保存するための最もシンプルな形式です。だからこそ、非常に広く使われています。ほとんどすべての文章作成ソフト、クラウドノート、オフィススイート、さらにはコマンドラインツールでも、テキストファイルを読み書きし、活用できます。

数十年前に DOS 上で作成されたテキストファイルは、現在でも Android で普通に読み書きできます。同じように、いま Android で書かれたテキストファイルも、これから数十年後の未来のコンピューターで読み書きできる可能性が非常に高いです。

このアプリは、あなたの文章を `.txt` のテキストファイルとして保存します。そのため、データの耐久性と移行のしやすさが高まります。将来、すべてのノートを別のクラウドノート、ナレッジベース、または別の執筆ツールへ移したくなっても、移行ははるかに簡単で、多くの場合は一度で行えます。
