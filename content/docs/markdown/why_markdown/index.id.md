Markdown adalah format menulis yang sederhana dan ringan. Markdown memungkinkan Anda menggunakan teks biasa untuk mengekspresikan banyak gaya rich text yang umum, seperti heading, penekanan, daftar, dan gambar, tanpa bergantung pada editor dokumen yang berat atau format dokumen rich text bergaya Word. Halaman ini sendiri juga ditulis dalam Markdown.

# Cara kerja sintaks Markdown

Markdown mudah dibaca bahkan sebelum dirender. Berikut beberapa contoh sintaks yang umum:

- `# Heading` membuat heading. Lebih banyak simbol `#` dapat digunakan untuk level heading yang lebih rendah.
- `---` membuat garis pemisah horizontal.
- `**bold**` membuat teks menjadi **tebal**.
- `*italic*` membuat teks menjadi *miring*.
- `![Name](Image URL link)` adalah sintaks untuk menyisipkan gambar.
- `- item` membuat item daftar.
- `~~text~~` menambahkan coretan, seperti ~~teks lama~~.

Karena sintaksnya sangat kecil dan langsung, Markdown membantu penulis tetap fokus pada konten, bukan pada alat pemformatan.

# Mengapa file teks

Teks biasa adalah format paling sederhana untuk menyimpan teks, dan justru itulah alasan format ini digunakan secara luas. Hampir semua aplikasi menulis, alat catatan cloud, paket perkantoran, bahkan alat baris perintah dapat membaca, menulis, dan menggunakan file teks.

File teks yang ditulis puluhan tahun lalu di sistem DOS masih bisa dibaca dan diedit secara normal di Android saat ini. Dengan cara yang sama, file teks yang ditulis di Android hari ini sangat mungkin tetap dapat dibaca dan ditulis oleh komputer masa depan beberapa dekade mendatang.

Aplikasi ini menyimpan tulisan Anda sebagai file teks `.txt`, sehingga data Anda lebih tahan lama dan lebih mudah dipindahkan. Jika suatu hari Anda ingin memindahkan semua catatan Anda ke buku catatan cloud lain, basis pengetahuan, atau alat menulis lain, proses migrasinya akan jauh lebih mudah dan sering kali dapat dilakukan dalam satu langkah.
