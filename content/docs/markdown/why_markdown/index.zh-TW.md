Markdown 是一種簡約、輕量的寫作格式。它允許你只用純文字來表達許多常見的富文字樣式，例如標題、強調、列表與圖片，而不必依賴笨重的文件編輯器或類似 Word 的富文字文件格式。本文本身也是使用 Markdown 撰寫的。

# Markdown 語法如何運作

Markdown 即使在渲染之前也很容易閱讀。下面是一些常見的語法範例：

- `# Heading` 用於建立標題。使用更多的 `#` 符號可以表示更低層級的標題。
- `---` 用於建立分隔線。
- `**bold**` 會讓文字顯示為 **粗體**。
- `*italic*` 會讓文字顯示為 *斜體*。
- `![Name](Image URL link)` 是插入圖片的語法。
- `- item` 用於建立列表項目。
- `~~text~~` 會加入刪除線，例如 ~~舊文字~~。

正因為它的語法非常簡潔直接，Markdown 能幫助寫作者把注意力放在內容本身，而不是排版工具上。

# 為什麼使用文字檔案

純文字是用來儲存文字內容最簡單的格式，也正因如此，它才會被如此廣泛地使用。幾乎所有寫作軟體、雲端筆記工具、辦公套件，甚至命令列工具，都可以讀寫和使用文字檔案。

數十年前在 DOS 系統中寫下的文字檔案，今天依然可以在 Android 上被正常讀取與編輯。同樣地，今天在 Android 上寫下的文字檔案，在數十年後的未來電腦上，也極有可能依然可以被讀取與寫入。

本應用會將你的寫作內容儲存為 `.txt` 文字檔案，這讓你的資料更耐久，也更容易遷移。如果未來某一天你想把所有筆記遷移到其他雲端筆記、知識庫或寫作工具中，整個遷移過程都會簡單得多，而且通常可以一步完成。
