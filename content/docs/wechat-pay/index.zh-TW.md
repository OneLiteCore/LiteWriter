# 啟用

一個啟用碼可以啟用 3 個專業版的副本，當你啟用第 4 個副本的時候舊副本將會被取消啟用。

啟用碼僅限個人使用。同時啟用碼是不記名且不可重置的，如果你洩露了啟用碼的話那麼任何獲得了啟用碼的人都能使用它，所以請將之儲存在一個安全的地方。

# 忘了你的啟用碼?

微信 -> 服務 -> 賬單 -> 查詢購物記錄 -> **商戶訂單號**

你可以長按並複製 **商戶訂單號** 並返回應用重新啟用你的應用專業版。
