WebDAV adalah protokol akses file yang banyak digunakan dan bisa menjadi pilihan yang fleksibel untuk cadangan cloud. Anda dapat menggunakan layanan WebDAV yang disediakan oleh beberapa penyedia penyimpanan cloud, atau Anda dapat menyiapkan server WebDAV Anda sendiri jika ingin memiliki kendali penuh atas tempat file cadangan Anda disimpan.

Bagi pengguna yang sangat peduli pada kepemilikan dan privasi, WebDAV bisa menjadi pilihan yang sangat menarik. Jika Anda menjalankan server sendiri, Anda dapat mempertahankan kendali penuh atas data cadangan Anda dari awal sampai akhir.
