WebDAV is a widely used file access protocol, and it can be a flexible choice for cloud backup. You can use a WebDAV service provided by some cloud storage vendors, or you can deploy your own WebDAV server if you want full control over where your backup files are stored.

For users who care deeply about ownership and privacy, WebDAV can be especially appealing. If you run your own server, you can keep complete control over your backup data from start to finish.
