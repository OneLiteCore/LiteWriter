WebDAV est un protocole d'accès aux fichiers largement utilisé, et il peut être un choix flexible pour la sauvegarde dans le cloud. Vous pouvez utiliser un service WebDAV proposé par certains fournisseurs de stockage cloud, ou déployer votre propre serveur WebDAV si vous souhaitez garder un contrôle total sur l'endroit où vos fichiers de sauvegarde sont stockés.

Pour les utilisateurs qui accordent une grande importance à la propriété des données et à la confidentialité, WebDAV peut être particulièrement intéressant. Si vous gérez votre propre serveur, vous pouvez garder un contrôle complet sur vos données de sauvegarde du début à la fin.
