WebDAV es un protocolo de acceso a archivos ampliamente utilizado, y puede ser una opción flexible para la copia de seguridad en la nube. Puedes usar un servicio WebDAV ofrecido por algunos proveedores de almacenamiento en la nube, o incluso desplegar tu propio servidor WebDAV si quieres tener control total sobre dónde se guardan tus archivos de respaldo.

Para quienes valoran mucho la propiedad y la privacidad, WebDAV puede resultar especialmente atractivo. Si administras tu propio servidor, puedes mantener un control completo sobre tus datos de copia de seguridad de principio a fin.
