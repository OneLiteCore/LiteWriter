WebDAV è un protocollo di accesso ai file ampiamente utilizzato e può essere una scelta flessibile per il backup nel cloud. Puoi usare un servizio WebDAV offerto da alcuni fornitori di archiviazione cloud, oppure puoi distribuire il tuo server WebDAV se vuoi avere il pieno controllo su dove vengono archiviati i tuoi file di backup.

Per gli utenti che tengono molto alla proprietà dei dati e alla privacy, WebDAV può essere particolarmente interessante. Se gestisci il tuo server, puoi mantenere il controllo completo dei tuoi dati di backup dall'inizio alla fine.
