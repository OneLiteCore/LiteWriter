title: Сервис резервного копирования в хранилище устройства
---

Сервис резервного копирования в хранилище устройства позволяет использовать папку на вашем устройстве как место для хранения резервных копий. После того как вы дадите приложению разрешение на чтение и запись для определённой папки в памяти устройства, приложение сможет сохранять туда файлы резервных копий.

Приложение может получать доступ только к той папке, которую вы выбрали для резервного копирования. Оно не может просматривать или открывать другие файлы в памяти вашего устройства.

# Эти данные тоже будут потеряны, если вы потеряете устройство

Если вы удалите приложение, файлы резервных копий, сохранённые в выбранной вами папке для резервного копирования в хранилище, удалены не будут. Это означает, что если вы позже снова установите приложение, то всё равно сможете восстановить данные из этой папки.

Однако эти файлы резервных копий по-прежнему хранятся на самом устройстве. Если устройство будет потеряно, украдено, серьёзно повреждено или его хранилище станет недоступным, эта часть ваших резервных данных тоже может быть потеряна.
