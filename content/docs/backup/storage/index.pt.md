title: Serviço de backup em armazenamento
---

O serviço de backup em armazenamento permite que você use uma pasta do seu dispositivo como local de backup. Depois de conceder ao aplicativo permissão de leitura e gravação para uma pasta específica no armazenamento do dispositivo, o aplicativo poderá salvar seus arquivos de backup ali.

O aplicativo só pode acessar a pasta que você escolher para o backup. Ele não pode navegar nem acessar outros arquivos no armazenamento do seu dispositivo.

# Esses dados também serão perdidos se você perder o dispositivo

Se você desinstalar o aplicativo, os arquivos de backup armazenados na pasta que você escolheu para o backup em armazenamento não serão excluídos. Isso significa que, se você instalar o aplicativo novamente mais tarde, ainda poderá restaurar seus dados a partir dessa pasta.

No entanto, esses arquivos de backup ainda ficam armazenados no próprio dispositivo. Se o dispositivo for perdido, roubado, seriamente danificado ou se o armazenamento dele ficar indisponível, essa parte dos seus dados de backup também poderá ser perdida.
