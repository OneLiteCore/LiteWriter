title: Servicio de copia de seguridad en almacenamiento
---

El servicio de copia de seguridad en almacenamiento te permite usar una carpeta de tu dispositivo como ubicación de respaldo. Después de conceder a la aplicación permiso de lectura y escritura para una carpeta específica dentro del almacenamiento del dispositivo, la aplicación podrá guardar allí tus archivos de copia de seguridad.

La aplicación solo puede acceder a la carpeta que elijas para la copia de seguridad. No puede explorar ni acceder a otros archivos del almacenamiento de tu dispositivo.

# Estos datos también se perderán si pierdes tu dispositivo

Si desinstalas la aplicación, los archivos de copia de seguridad almacenados en la carpeta que elegiste para la copia en almacenamiento no se eliminarán. Esto significa que, si vuelves a instalar la aplicación más adelante, todavía podrás restaurar tus datos desde esa carpeta.

Sin embargo, estos archivos de copia de seguridad siguen almacenados en el propio dispositivo. Si el dispositivo se pierde, es robado, sufre daños graves o su almacenamiento deja de estar disponible, esta parte de tus datos de copia de seguridad también podría perderse.
