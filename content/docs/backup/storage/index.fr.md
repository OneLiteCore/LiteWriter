title: Service de sauvegarde sur stockage
---

Le service de sauvegarde sur stockage vous permet d'utiliser un dossier de votre appareil comme emplacement de sauvegarde. Après avoir accordé à l'application l'autorisation de lecture et d'écriture pour un dossier précis dans le stockage de votre appareil, l'application peut y enregistrer vos fichiers de sauvegarde.

L'application ne peut accéder qu'au dossier que vous choisissez pour la sauvegarde. Elle ne peut ni parcourir ni accéder aux autres fichiers présents dans le stockage de votre appareil.

# Ces données seront également perdues si vous perdez votre appareil

Si vous désinstallez l'application, les fichiers de sauvegarde stockés dans le dossier que vous avez choisi pour la sauvegarde sur stockage ne seront pas supprimés. Cela signifie que, si vous réinstallez l'application plus tard, vous pourrez toujours restaurer vos données depuis ce dossier.

Cependant, ces fichiers de sauvegarde restent stockés sur l'appareil lui-même. Si l'appareil est perdu, volé, gravement endommagé ou si son stockage devient indisponible, cette partie de vos données de sauvegarde peut également être perdue.
