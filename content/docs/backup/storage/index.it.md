title: Servizio di backup su archiviazione
---

Il servizio di backup su archiviazione ti permette di usare una cartella del tuo dispositivo come posizione di backup. Dopo aver concesso all'app il permesso di leggere e scrivere in una cartella specifica nella memoria del dispositivo, l'app potrà salvare lì i tuoi file di backup.

L'app può accedere solo alla cartella che scegli per il backup. Non può sfogliare né accedere agli altri file presenti nella memoria del dispositivo.

# Anche questi dati andranno persi se perdi il dispositivo

Se disinstalli l'app, i file di backup archiviati nella cartella che hai scelto per il backup su archiviazione non verranno eliminati. Questo significa che, se reinstalli l'app in un secondo momento, potrai comunque ripristinare i tuoi dati da quella cartella.

Tuttavia, questi file di backup restano comunque memorizzati sul dispositivo stesso. Se il dispositivo viene smarrito, rubato, gravemente danneggiato o il suo spazio di archiviazione non è più disponibile, anche questa parte dei tuoi dati di backup potrebbe andare persa.
