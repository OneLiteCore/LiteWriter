title: Perkhidmatan sandaran storan
---

Perkhidmatan sandaran storan membolehkan anda menggunakan satu folder pada peranti anda sebagai lokasi sandaran. Selepas anda memberikan kebenaran baca dan tulis kepada aplikasi untuk folder tertentu dalam storan peranti anda, aplikasi boleh menyimpan fail sandaran anda di sana.

Aplikasi hanya boleh mengakses folder yang anda pilih untuk sandaran. Ia tidak boleh menyemak imbas atau mengakses fail lain dalam storan peranti anda.

# Data ini juga akan hilang jika anda kehilangan peranti anda

Jika anda menyahpasang aplikasi, fail sandaran yang disimpan dalam folder yang anda pilih untuk sandaran storan tidak akan dipadamkan. Ini bermakna jika anda memasang semula aplikasi itu kemudian, anda masih boleh memulihkan data anda daripada folder tersebut.

Walau bagaimanapun, fail sandaran ini masih disimpan pada peranti itu sendiri. Jika peranti itu hilang, dicuri, rosak teruk, atau storannya tidak lagi tersedia, bahagian data sandaran anda ini juga mungkin akan hilang.
