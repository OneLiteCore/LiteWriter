title: Serbisyo ng backup sa storage
---

Hinahayaan ka ng serbisyo ng backup sa storage na gumamit ng isang folder sa iyong device bilang lokasyon ng backup. Kapag binigyan mo ang app ng pahintulot na magbasa at magsulat sa isang partikular na folder sa storage ng iyong device, maaari nitong i-save roon ang iyong mga backup file.

Ang app ay maaari lamang mag-access sa folder na pinili mo para sa backup. Hindi nito mabubuksan o maa-access ang iba pang mga file sa storage ng iyong device.

# Mawawala rin ang mga datos na ito kapag nawala ang iyong device

Kung i-uninstall mo ang app, hindi mabubura ang mga backup file na nakaimbak sa folder na pinili mo para sa storage backup. Ibig sabihin, kung muli mong i-install ang app sa ibang pagkakataon, maaari mo pa ring ibalik ang iyong data mula sa folder na iyon.

Gayunpaman, ang mga backup file na ito ay nakaimbak pa rin mismo sa device. Kung mawala ang device, manakaw, masira nang husto, o hindi na maging available ang storage nito, maaari ring mawala ang bahaging ito ng iyong backup data.
