title: Layanan cadangan penyimpanan
---

Layanan cadangan penyimpanan memungkinkan Anda menggunakan folder di perangkat Anda sebagai lokasi cadangan. Setelah Anda memberikan izin baca dan tulis kepada aplikasi untuk folder tertentu di penyimpanan perangkat, aplikasi dapat menyimpan file cadangan Anda di sana.

Aplikasi hanya dapat mengakses folder yang Anda pilih untuk cadangan. Aplikasi tidak dapat menelusuri atau mengakses file lain di penyimpanan perangkat Anda.

# Data ini juga akan hilang jika Anda kehilangan perangkat

Jika Anda menghapus aplikasi, file cadangan yang disimpan di folder yang Anda pilih untuk cadangan penyimpanan tidak akan dihapus. Ini berarti jika Anda memasang aplikasi lagi nanti, Anda tetap dapat memulihkan data dari folder tersebut.

Namun, file cadangan ini tetap disimpan di perangkat itu sendiri. Jika perangkat hilang, dicuri, rusak parah, atau penyimpanannya tidak lagi tersedia, bagian dari data cadangan Anda ini juga dapat hilang.
